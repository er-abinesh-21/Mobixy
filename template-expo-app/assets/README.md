# Assets Directory

This folder contains the app's visual assets.

## Required Files

| File | Size | Purpose |
|------|------|---------|
| `icon.png` | 1024x1024 | Main app icon |
| `adaptive-icon.png` | 1024x1024 | Android adaptive icon foreground |
| `splash.png` | 1284x2778 | Splash screen |
| `favicon.png` | 32x32 | Web favicon |

## Notes

- All icons should be **PNG format**
- Use a **square** image for best results
- Icons are automatically replaced during the build process
- The splash screen is generated with the icon centered on a gradient background

## Placeholder Images

The placeholder images in this folder are replaced during each build with the user's custom icon.
